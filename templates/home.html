Using Image Tool API to apply image filtering

There is only 1 route

POST /

The body should be a multi-part form with certain key value pairs.

Example:

POST /
image : example.jpg
filter : find_edges

All request must have an 'image' key with the actual image file as the value and and a 'filter' key with the filter function name as a the value. But, certain filters require additional parameters. These parameters should be specified as additional key value pairs.

Example:

POST /
image : example.jpg
filter : unsharp_mask
radius : 2
percent : 150
threshold : 1
